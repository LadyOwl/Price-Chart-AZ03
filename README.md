# 📊 Price-Chart-AZ03

**Анализ данных, визуализация и парсинг цен на Python**

Набор из трёх упражнений, демонстрирующих навыки работы с данными: от генерации случайных чисел и построения графиков до парсинга реальных цен с интернет-магазина и их статистического анализа.

---

## 🎯 О проекте

Price-Chart-AZ03 — это учебный проект, созданный для освоения ключевых инструментов анализа данных на Python. Проект включает три последовательно усложняющихся упражнения: от простой гистограммы до полноценного парсера с анализом реальных данных.

### 💡 Ключевые особенности

- ✅ **Три уровня сложности** — от базовой визуализации до веб-парсинга
- ✅ **NumPy** — генерация и обработка числовых данных
- ✅ **Matplotlib** — построение гистограмм и диаграмм рассеяния
- ✅ **Pandas** — загрузка, очистка и анализ CSV-данных
- ✅ **Selenium WebDriver** — автоматизация браузера для парсинга
- ✅ **Реальные данные** — парсинг цен с сайта divan.ru
- ✅ **Статистический анализ** — расчёт средней цены, фильтрация данных

## 💻 Ключевые фрагменты кода

### Гистограмма нормального распределения (exercise1.py)

```python
import numpy as np
import matplotlib.pyplot as plt

# Параметры нормального распределения
mean = 0          # Среднее значение
std_dev = 1       # Стандартное отклонение
num_samples = 1000

# Генерация данных
data = np.random.normal(mean, std_dev, num_samples)

# Построение гистограммы
plt.figure(figsize=(10, 6))
plt.hist(data, bins=30, color='skyblue', edgecolor='black', alpha=0.7)
plt.title('Гистограмма случайных данных (нормальное распределение)')
plt.xlabel('Значение')
plt.ylabel('Частота')
plt.grid(True, alpha=0.3)
plt.show()
```

### Парсинг цен с сайта (exercise3.py)

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
import re

def get_prices():
    driver = webdriver.Firefox()
    driver.get("https://www.divan.ru/category/divany?sort=4")
    time.sleep(5)  # ждем подгрузку контента
    
    prices = []
    price_elements = driver.find_elements(By.CSS_SELECTOR, "span.ui-LD-ZU")
    
    for elem in price_elements:
        text = elem.text.strip()
        match = re.search(r"(\d[\d\s]*)", text)
        if match:
            num = int(match.group(1).replace(" ", ""))
            prices.append(num)
    
    driver.quit()
    return prices
```

### Анализ данных через Pandas (exercise3.py)

```python
import pandas as pd
import matplotlib.pyplot as plt

def load_and_analyze(filename="divan_prices.csv"):
    df = pd.read_csv(filename)
    df["price"] = pd.to_numeric(df["price"], errors="coerce")
    df = df.dropna(subset=["price"])
    
    mean_price = df["price"].mean()
    print(f"Средняя цена дивана: {mean_price:.2f} руб.")
    
    # Гистограмма цен
    plt.figure(figsize=(10, 6))
    plt.hist(df["price"], bins=30, color='skyblue', edgecolor='black')
    plt.title("Гистограмма цен на диваны (руб.)")
    plt.xlabel("Цена, руб.")
    plt.ylabel("Количество моделей")
    plt.grid(axis='y', alpha=0.75)
    plt.tight_layout()
    plt.show()
```
